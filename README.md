# 💻 Online Coding Platform

A full-stack online coding platform that allows users to write, run, and test code in multiple programming languages using the Judge0 API.

---

## 🚀 Features

- 🧑‍💻 Interactive code editor
- ⚡ Real-time code execution
- 🌐 Supports multiple programming languages
- 🔗 API-based execution using Judge0
- 🖥️ Clean and responsive UI
- 📡 Internet-based platform (works anywhere)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Realtime (optional):** Socket.IO
- **Code Execution API:** Judge0
